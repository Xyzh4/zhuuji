# 疆来筑迹 - 文旅互动体验平台

## 项目简介

本项目是一个关于新疆民族文化的文旅互动体验平台，旨在弘扬新疆民族文化，促进非遗活态传承。平台包含多个模块，展示新疆的地貌、民俗、节日、美食、旅游等内容。

## 项目结构

```
├── .idea/              # IDE 配置文件
├── .vscode/            # VS Code 配置文件
├── backend/            # 后端代码
│   ├── java/           # Java 后端
│   └── python/         # Python 工具脚本
├── *.html              # 前端页面
├── script.js           # 前端脚本
├── style.css           # 前端样式
├── *.png, *.jpg        # 图片文件
├── start_server.bat    # 启动脚本
└── README.md           # 本说明文件
```

## 如何运行

### 方法一：使用本地服务器（推荐）

1. 确保您的计算机上安装了 Python
2. 双击 `start_server.bat` 文件启动本地服务器
3. 服务器启动后，在浏览器中访问 `http://localhost:8000`
4. 开始浏览和体验平台内容

### 方法二：直接打开 HTML 文件

1. 找到 `index.html` 文件
2. 双击打开该文件
3. 注意：由于浏览器的安全限制，某些功能可能无法正常使用

## 主要功能模块

- **疆来·阿达西** (home.html)：平台首页，介绍新疆概况
- **疆蕴·观地貌** (landscape.html)：展示新疆的地形地貌，包含山脉和盆地的图表
- **疆俗·看民俗** (ethnic.html)：介绍新疆各民族的风俗习惯
- **疆历·过节日** (timeline.html)：展示新疆各民族的传统节日
- **疆味·香满口** (food.html)：介绍新疆的特色美食
- **疆域·逛南北** (map.html)：展示新疆的地理分布
- **疆行·品游记** (travel.html)：展示新疆的旅游景点和游记

## 技术说明

- **前端**：HTML5, CSS3, JavaScript, Chart.js, ECharts
- **后端**：Java Spring Boot, JPA, H2/MySQL
- **数据**：使用 H2 内存数据库存储民族数据

## 注意事项

1. 项目使用了一些外部资源，如图片和音乐
2. 如果外部资源无法加载，可能会影响部分功能的显示效果
3. 建议使用方法一运行项目，以获得最佳体验
4. 后端功能需要 Java 环境支持，如果仅需前端功能，可忽略后端部分

## 故障排除

- **页面无法打开**：确保本地服务器已启动，并且在浏览器中输入了正确的地址
- **图表无法显示**：检查浏览器控制台是否有错误信息，可能是 Chart.js 加载失败
- **图片无法加载**：可能是网络连接问题，或外部图片 API 暂时不可用
