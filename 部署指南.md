# 新疆旅游网站 - 部署指南

## 🚀 最简单的部署方案（推荐）

### 方法一：Netlify Drop（5分钟搞定！）

1. 访问：https://app.netlify.com/drop
2. 把整个 `5,1` 文件夹拖进去
3. 等几秒钟，就获得一个网站链接！

**优点**：超级简单，不需要任何技术知识
**注意**：只显示静态页面，API接口不会工作

---

### 方法二：Vercel（推荐，免费又快）

1. 访问：https://vercel.com/
2. 用 GitHub 账号登录
3. 点击 "Add New..." → "Project"
4. 点击 "Browse" 上传文件夹
5. 选择您的 `5,1` 文件夹
6. 点击 "Deploy"

---

## 🛠️ 如果您想用完整功能（需要Git）

### 第一步：安装 Git

1. 下载：https://git-scm.com/download/win
2. 安装（一路点击 Next）
3. 重启终端
4. 验证：`git --version`

### 第二步：创建 GitHub 仓库

1. 访问：https://github.com/new
2. 仓库名：`xinjiang-tourism`
3. 点击 "Create repository"

### 第三步：上传代码

```powershell
cd c:\Users\杨子涵\Desktop\5,1
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/您的用户名/xinjiang-tourism.git
git push -u origin main
```

### 第四步：使用 Render 部署

1. 访问：https://render.com/
2. 用 GitHub 登录
3. 点击 "New +" → "Web Service"
4. 选择您的仓库
5. 配置：
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `gunicorn --workers=4 app:app`
6. 点击 "Create Web Service"

---

## 📝 注意事项

- 如果不需要数据库功能，用 Netlify/Vercel 最简单
- 如果需要完整功能（API接口），需要用 Render + 云端数据库
- 云端数据库推荐：PlanetScale、Railway MySQL
